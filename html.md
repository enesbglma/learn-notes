# HTML Temelleri – Notlarım

## 1. Temel Etiketler
- `<p>` : Paragraf
- `<h1>` - `<h6>` : Başlıklar
- `<a>` : Link
- `<img>` : Görsel eklemek

## 2. Örnek Sayfa
```html
<!DOCTYPE html>
<html>
  <head>
    <title>İlk HTML Sayfam</title>
  </head>
  <body>
    <h1>Merhaba Dünya!</h1>
    <p>Bu benim ilk HTML notum.</p>
  </body>
</html>
```

## 3. Öğrenilecekler
- Formlar (`<form>`, `<input>`, `<button>`)
- Listeleme (`<ul>`, `<ol>`, `<li>`)
- Semantik etiketler (`<header>`, `<footer>`, `<main>`)
